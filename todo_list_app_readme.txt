# Todo List App

This is a simple Todo List App created with HTML, CSS, and JavaScript.

## Features

- Add new todos
- Remove todos
- Mark todos as completed
- Edit existing todos
- View and delete completed todos

## How to Use

1. Enter a new todo in the input field.
2. Click "Add Todo" to add the todo to the list.
3. Click on a todo to mark it as completed.
4. Click "Edit" to modify a todo.
5. Click "Remove" to delete a todo.
6. View completed todos in the "Completed Todos" section and delete them if needed.
